## ⚡ Consumo de Energia - Eletrodoméstico


## 🎯 Objetivo

Este programa tem como objetivo fazer uma calculadora para consumo de energia elétrica.


## 💻 Linguagem utilizada

🐍 Desenvolvida em Python.


## 📚 Aprendizados

Permite que o usuário informe:

- Nome do eletrodoméstico
- Potência do aparelho
- Horas de uso por dia
- Conversão para consumo mensal
- Valor do Kwh (R$)
- Cálculo final do gasto de energia em kWh e em reais.

💡 O sistema colabora para entender quanto um aparelho consome de energia elétrica, facilitando o controle de gastos na conta de luz.

---

O projeto foi desenvolvido para prática da lógica de programação, entrada de dados e operações matemáticas.
